# Markdown 基础与常用语法

Markdown 是一种轻量级标记语言，可以使用简单的符号快速编写结构清晰的文档。

## 1. 标题

使用 `#` 表示标题，`#` 的数量代表标题级别，最多支持六级标题。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 2. 段落与换行

段落之间需要保留一个空行。

在一行末尾输入两个空格再换行，或者使用 `<br>`，可以在同一段内强制换行。

```markdown
这是第一段。

这是第二段。

第一行  
第二行
```

## 3. 文本样式

```markdown
*斜体文本*
_斜体文本_

**粗体文本**
__粗体文本__

***粗斜体文本***

~~删除线文本~~

<u>下划线文本</u>
```

显示效果：

- *斜体文本*
- **粗体文本**
- ***粗斜体文本***
- ~~删除线文本~~

> 注意：下划线和删除线并非最基础的 Markdown 标准，部分编辑器可能不支持。

## 4. 引用

在文本前添加 `>` 创建引用。引用可以嵌套。

```markdown
> 这是一段引用。
>
> > 这是嵌套引用。
```

## 5. 列表

### 无序列表

使用 `-`、`*` 或 `+` 创建无序列表。

```markdown
- 苹果
- 香蕉
  - 小香蕉
  - 大香蕉
- 橙子
```

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 6. 链接

```markdown
[链接文字](https://example.com)
[带提示的链接](https://example.com "鼠标悬停提示")
<https://example.com>
```

也可以使用引用式链接：

```markdown
[示例网站][example]

[example]: https://example.com "示例网站"
```

## 7. 图片

图片语法与链接相似，只是在开头多一个 `!`。

```markdown
![图片说明](images/example.png)
![网络图片](https://example.com/image.png "可选标题")
```

## 8. 代码

### 行内代码

使用反引号包裹行内代码：

```markdown
使用 `npm install` 安装依赖。
```

### 代码块

使用三个反引号包裹代码，并在开头标明语言以启用语法高亮。

````markdown
```javascript
function hello() {
  console.log("Hello, Markdown!");
}
```
````

## 9. 分隔线

单独一行输入三个或更多的 `-`、`*` 或 `_`。

```markdown
---
***
___
```

## 10. 表格

使用竖线分隔列，使用 `:` 设置对齐方式。

```markdown
| 姓名 | 年龄 | 城市 |
| :--- | ---: | :---: |
| 张三 | 20 | 北京 |
| 李四 | 25 | 上海 |
```

- `:---`：左对齐
- `---:`：右对齐
- `:---:`：居中对齐

## 11. 转义字符

如果想显示 Markdown 本身使用的特殊符号，可以在符号前添加反斜杠 `\`。

```markdown
\*这段文字不会变成斜体\*
\# 这行文字不会变成标题
```

常见的可转义字符包括：

```text
\  `  *  _  { }  [ ]  ( )  #  +  -  .  !  |  >
```

## 12. 脚注

部分 Markdown 编辑器支持脚注：

```markdown
这是一段带脚注的文字。[^1]

[^1]: 这是脚注内容。
```

## 13. 折叠内容（HTML 扩展）

Markdown 中通常可以混用 HTML：

```html
<details>
  <summary>点击查看详情</summary>

  这里是折叠的内容。
</details>
```

## 14. 常用组合示例

````markdown
# 项目名称

这是项目的简要介绍。

## 功能

- [x] 用户登录
- [ ] 数据导出

## 安装

```bash
npm install
npm run dev
```

更多内容请查看[项目文档](https://example.com)。
````

## 使用提示

- 标记符号后通常需要保留一个空格，例如 `# 标题` 和 `- 列表项`。
- 嵌套列表通常缩进两个或四个空格，具体效果取决于渲染器。
- 不同平台对扩展语法的支持可能不同，表格、任务列表和脚注尤其需要注意兼容性。
- 编写完成后，建议在目标平台预览最终渲染效果。
