# MySQL 基础概念

## 1. 数据库与数据库管理系统

数据库是按照一定结构组织和保存数据的集合。数据库管理系统（Database Management System，DBMS）是用来创建、查询、修改和管理数据库的软件。

MySQL 是一个关系型数据库管理系统（Relational Database Management System，RDBMS）。关系型数据库主要使用表来保存数据，并通过表之间的关联表达数据关系。

例如，一个学生管理系统可以包含：

- `students` 表：保存学生信息。
- `courses` 表：保存课程信息。
- `student_courses` 表：保存学生的选课记录。

## 2. 数据库的层级结构

MySQL 中的数据可以简化为以下层级：

```text
MySQL 服务器
├── 数据库 A
│   ├── 表 1
│   └── 表 2
└── 数据库 B
    └── 表 3
```

## 3. SQL 的主要分类

SQL（Structured Query Language，结构化查询语言）是操作关系型数据库的语言。常见 SQL 可以按用途分为：

| 分类 | 作用 |
| ---- | ---- |
| DDL，数据定义语言 | 定义数据库和表的结构 |
| DML，数据操作语言 | 新增、修改和删除数据 |
| DQL，数据查询语言 | 查询数据 |
| DCL，数据控制语言 | 创建数据库用户，控制用户的访问权限 |
| TCL，事务控制语言 | 管理事务 |

SQL 语句中的关键字不区分大小写，但通常使用大写关键字、小写对象名，以提高可读性。
